# 🔐 Генератор паролей

Простая программа на Java для генерации случайных паролей (длина, символы, цифры, буквы).

## Возможности

- Выбор длины пароля
- Добавление спецсимволов (`!@#$%&*`)
- Добавление цифр
- Случайная генерация

## Как запустить

1. Скопируйте код в свою IDE
2. Запустите `PasswordGenerator.java`
3. Следуйте инструкциям в консоли

## Пример работы

=== Генератор паролей ===

Введите длину пароля: 12

Использовать спецсимволы? (да/нет): да

Использовать цифры в пароле? (да/нет): да

Ваш пароль: aB3$xK9@mP2!
